# stock-portfolio — Issue Tracker

This repository is used to track public issues for the [stock-portfolio](https://github.com/namila007/stock-portfolio) project.

---

## Reporting an Issue

Click **[New Issue](../../issues/new/choose)** and select the appropriate template. Blank issues are disabled — please use one of the templates below so that every report contains the context needed to investigate and fix it quickly.

---

## Issue Templates

### 🐛 Bug Report

Use this template to report unexpected behaviour, errors, or exceptions in the stock-portfolio application.

| Field | Required | Description |
|---|:---:|---|
| **Username** | No | The username or account affected by the issue. |
| **Session / Correlation ID** | No | Session or correlation ID from application logs or response headers — used to trace the exact request. |
| **Exception / Error Details** | No | Full exception message, stack trace, or error output (rendered as a code block). |
| **Description** | ✅ | A clear and concise description of what the bug is. |
| **Steps to Reproduce** | ✅ | Step-by-step instructions to reproduce the issue. |
| **Expected Behavior** | No | What you expected to happen. |
| **Actual Behavior** | No | What actually happened. |
| **Severity** | No | `Critical` / `High` / `Medium` / `Low` |
| **Environment** | No | Where the issue occurred, e.g. Production, Staging, Local. |
| **Additional Context** | No | Any extra screenshots, log snippets, or notes. |

### ✨ Feature Request

Use this template to suggest a new feature or enhancement.

| Field | Required | Description |
|---|:---:|---|
| **Problem Statement** | ✅ | The problem or limitation you are facing. |
| **Proposed Solution** | ✅ | The feature or change you would like to see. |
| **Alternatives Considered** | No | Other approaches or workarounds you have tried. |
| **Additional Context** | No | Mockups, links, or other supporting information. |

---

## Labels

| Label | Description |
|---|---|
| `bug` | Automatically applied to all Bug Report issues. |
| `enhancement` | Automatically applied to all Feature Request issues. |
